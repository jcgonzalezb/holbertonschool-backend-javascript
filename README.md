# Back-end - Modern JavaScript

Each folder in this repository illustrates an essential concept of programming, specific to the JavaScript language. The file names reference the concepts. Inside these folders are files containing JavaScript scripts and functions around that particular concept.

- 0x00. ES6 Basics
- 0x01. ES6 Promises
- 0x02. ES6 classes
- 0x03. ES6 data manipulation
- 0x04. Typescript
- 0x05. NodeJS Basics
- 0x06. Unittests in JS

## Languages and Tools:

- OS: Ubuntu 20.04 LTS
- Language: Javascript (ES6), Node v14.
- Style guidelines: [Javascript Standard](https://standardjs.com/rules.html) || [ESLint](https://eslint.org/)
